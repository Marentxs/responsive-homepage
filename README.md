A fully responsive developer portfolio showcasing my front-end projects, built with HTML and CSS. With dynamic layouts that adapt from mobile devices desktop to desktop.

## Features

- **Fully Responsive Design** - Three distinct breakpoints (2560px, 1024px, 640px) for responsiveness on all devices
- **Retro Aesthetic**
- **Interactive Elements** - Cards and icons show hover effects

## Built With

- HTML5 (Semantic markup with accessibility considerations)
- CSS3 (Flexbox, Grid, Custom properties)
- Responsive Design (Mobile-first approach with media queries)
- Accessibility (ARIA labels, semantic landmarks, skip navigation ready)
